### behaviour_adjustment:
update agent behaviour per user request
write instructions to add or remove to adjustments arg
usage:
```xml
<response>
  <thoughts>Adjusting behavior per user request.</thoughts>
  <headline>Adjusting agent behavior per user request</headline>
  <tool_name>behaviour_adjustment</tool_name>
  <tool_args>
    <adjustments>remove...</adjustments>
  </tool_args>
</response>
```
