# 我县举办全民健身新春赛事嘉...

**图片说明/字幕：**
侯志福 少年强运动中心总经理
在“38”女神节期间举办赛事嘉年华的活动

**正文内容：**

少年强运动中心总经理 侯志福：在“38”女神节期间举办赛事嘉年华的活动，一来我们是为广大体育爱好者搭建交流展示的平台，提升大家的身体素质和竞技水平，二来为黄冈市第七届运动会做赛前的动员以及选才做预热，为大家能够提供赛事级的场馆服务。我由衷地感到骄傲。

**图片内容（场馆内景）：**
（背景横幅可见文字：以体树人、一起上场）

**正文内容：**

据了解，少年强运动中心，位于天泽大酒店区域，建筑面积约4000平方米，内含羽毛球场、篮球场、气排球场、少儿体能馆、搏击馆等多类专业化运动空间，并配备更衣淋浴、智能管理等服务设施。全年运营可服务群众健身超8万人次，将为我县“体育+文旅”融合发展贡献力量。
