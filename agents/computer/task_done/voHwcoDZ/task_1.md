## 正文内容

可能是周末的午后，阳光透过窗棂洒在桌角，你煮了一壶喜欢的茶，翻开一本搁置许久的书。

指尖划过书页，茶香漫过鼻尖，时间慢得刚刚好。

可能是雨后的黄昏，你推开窗，空气里满是湿润的泥土和青草的气息。

楼下的花坛边，几朵不知名的小花正仰着脸，花瓣上还挂着晶莹的雨珠。

生活的诗意从来不在远方，就在这些不起眼的角落里，等着被某个停下来的人看见。

那一刻，心里会有一个声音轻轻响起：这样，就很好。

“活在当下，便是人生最好的修行。”

我们总想着抓住整个人生，却常常在追逐中，漏掉了最珍贵的此刻。

幸福就在当下。
