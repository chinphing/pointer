# 中国人民银行副行长宣昌能出席“亚洲2050”国际会议

**来源：** 中国人民银行
**发布时间：** 2026年3月9日 21:23
**地点：** 北京

---

2026年3月5日至6日，国际货币基金组织与泰国央行联合举办“亚洲2050”国际会议。会议聚焦亚洲未来经济增长前景、人口结构与技术转型以及中央银行政策应对等问题，中国人民银行副行长宣昌能应邀出席并就“亚洲中央银行面临的挑战”议题发言。

宣昌能指出，新兴市场经济体面临多重机遇与挑战，人口结构变化、技术进步以及地缘政治等因素对经济增长、就业和金融稳定产生深远影响。亚洲各国中央银行应致力于提升国内经济韧性，完善货币政策框架，加强宏观经济政策协调与合作，携手推进全球经济治理改革，更好应对外部冲击。中方愿继续深化与各方合作，为促进区域和全球经济稳定增长作出积极贡献。

宣昌能表示，中国增长动能稳健，科技创新活跃，新质生产力加快发展，经济结构不断转型升级，就业总体保持平稳，人民币汇率在合理均衡水平上保持基本稳定，中国经济展现出较强韧性。中国持续推进基础设施绿色和数字化转型升级，提升了全球绿色技术与数字产品的可及性和可负担性，为全球绿色和数字化转型提供了重要的公共产品。中国人民银行坚持实施好适度宽松的货币政策，为经济持续向好创造适宜的货币金融环境。同时持续完善货币政策框架，更加注重发挥利率等价格型调控工具的作用，进一步健全市场化的利率调控机制，丰富货币政策工具箱，健全包括维护资本市场稳定安排在内的结构性货币政策工具体系，研究建立特定情形下对非银机构的流动性支持机制。
