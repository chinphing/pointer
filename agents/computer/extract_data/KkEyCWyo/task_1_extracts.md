# 我县举办全民健身新春赛事嘉年华活动

**发布账号**：秀美浠水

![图片：室内气排球比赛现场，多名运动员正在网前竞技，背景悬挂红色横幅“专注体育教育，助力年轻一代身心健康和全面发展”](image1.jpg)

活动由浠水县全民健身指导中心指导，县气排球协会、羽毛球协会、篮球协会联合主办，少年强运动中心协办。本次嘉年华整合三大赛事，涵盖不同群体与项目：女神节气排球友谊赛于3月6日至8日展开，12支女子代表队以分组循环+淘汰赛形式竞技；羽毛球混合团体争霸赛同期举行，10支混合团体通过男双、女双、混双等项目切磋技艺；男子篮球联赛则贯穿整个活动周期，12支代表队以分组循环+交叉淘汰模式展开较量。
