### input:
use keyboard arg for terminal program input
use session arg for terminal session number
answer dialogues enter passwords etc
not for browser
usage:
```xml
<response>
  <thoughts>The program asks for Y/N...</thoughts>
  <headline>Responding to terminal program prompt</headline>
  <tool_name>input</tool_name>
  <tool_args>
    <keyboard>Y</keyboard>
    <session>0</session>
  </tool_args>
</response>
```
